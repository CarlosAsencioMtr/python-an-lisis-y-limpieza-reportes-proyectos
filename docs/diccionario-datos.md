# Diccionario de Datos

Este documento describe los campos utilizados en el archivo `proyectos_ejemplo.csv`.

| Campo | Descripción | Tipo de dato |
|---|---|---|
| codigo_proyecto | Código único del proyecto | Texto |
| nombre_proyecto | Nombre ficticio del proyecto | Texto |
| direccion | Dirección o área relacionada con el proyecto | Texto |
| responsable | Responsable ficticio del proyecto | Texto |
| estado | Estado actual del proyecto | Texto |
| fecha_inicio | Fecha de inicio del proyecto | Fecha |
| fecha_fin_estimada | Fecha estimada de finalización | Fecha |
| fecha_fin_real | Fecha real de finalización | Fecha |
| porcentaje_avance | Porcentaje de avance del proyecto | Número |
| presupuesto | Presupuesto ficticio del proyecto | Número |
| ultima_actualizacion | Fecha de última actualización del registro | Fecha |

## Nota

Los datos utilizados son completamente ficticios y tienen fines demostrativos.
