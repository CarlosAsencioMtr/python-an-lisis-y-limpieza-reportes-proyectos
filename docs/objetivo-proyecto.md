# Objetivo del Proyecto

El objetivo de este proyecto es demostrar el uso de Python para automatizar el análisis de reportes de proyectos.

El proyecto toma datos ficticios de proyectos, realiza limpieza, validación, cálculo de KPIs, generación de gráficos y exportación de un reporte final en Excel.

## Objetivos específicos

- Cargar datos desde un archivo CSV.
- Limpiar y transformar la información.
- Validar calidad de datos.
- Calcular indicadores clave.
- Identificar proyectos atrasados.
- Generar gráficos de análisis.
- Exportar resultados a Excel y CSV.
