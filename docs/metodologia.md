# Metodología

El proyecto sigue una metodología simple de análisis de datos con Python.

## 1. Carga de datos

Los datos se cargan desde un archivo CSV ubicado en la carpeta `data`.

## 2. Limpieza de datos

Se normalizan los nombres de columnas, se eliminan espacios innecesarios, se convierten fechas y se transforman campos numéricos.

## 3. Validación

Se revisan valores nulos, duplicados, porcentajes fuera de rango y registros incompletos.

## 4. Análisis

Se calculan KPIs relacionados con proyectos activos, finalizados, nuevos, atrasados, avance promedio y presupuesto total.

## 5. Visualización

Se generan gráficos para visualizar proyectos por estado y avance promedio por dirección.

## 6. Exportación

Los resultados se exportan a Excel y CSV para facilitar su uso en reportes o dashboards.
